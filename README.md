##  What is NLP?

Natural Language Processing (NLP) is a branch of AI that enables machines to **understand, interpret, and generate human language**.

It bridges the gap between **human communication** and **computer understanding**.

---

##  Types of NLP

1. **Natural Language Understanding (NLU)**
   - Focuses on understanding text meaning  
   - Example: sentiment detection  

2. **Natural Language Generation (NLG)**
   - Generates human-like text  
   - Example: chatbots, text summarization

##  Computer Vision with OpenCV

###  What is OpenCV?

OpenCV (Open Source Computer Vision Library) is an open-source library used for **real-time computer vision and image processing**.

It allows machines to **analyze, process, and understand visual data** like images and videos.

---

###  Applications of OpenCV

-  Face Detection & Recognition  
-  Object Detection (Self-driving cars)  
-  Image Processing  
-  Medical Image Analysis  
-  Motion Detection & Surveillance  
-  Augmented Reality
